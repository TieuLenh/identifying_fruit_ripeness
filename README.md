# Set up
Đứng tại thư mục gốc, chạy powershell(chú ý: phải là powershell để tránh lỗi nhất khi kích hoạt môi trường ảo cho python)
lệnh: pip -m venv venv

tiếp theo chạy lệnh: pip install -r requirements.txt (để tải các module cần thiết cho cả traing và chạy BE srever + predict)

# RUN 
cuối cùng chạy lệnh: python ./BE/server.py (hoặc click đến file này mà chạy)
